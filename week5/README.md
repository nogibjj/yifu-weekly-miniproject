# Mini Project 4

## Description

This is a simple project that converts between a ASCII string and Morse code.

## Steps to run

- `make format` to format code
  
- `make lint` to lint
  
- `cargo run` to run the program
  - Then you will get

```rust
Enter a string to convert:
Go Duke
--. ---   -.. ..- -.- .
Enter Morse code to convert:
--. ---   -.. ..- -.- .
GO  DUKE
```
